# Инструкция по использованию Git на Linux

Git — это распределённая система контроля версий, которая позволяет отслеживать изменения в файлах и управлять версиями проектов. Ниже приведены основные шаги и команды для работы с Git на Linux.

---

## 1. Установка Git
Чтобы установить Git на Linux, выполните следующие команды в терминале:

**Для Debian/Ubuntu:**

sudo apt update
sudo apt install git


**Как склонить репу?**
Для того чтобы склонировать репозиторий, первое, что нужно сделать это обменяться с *git* - **ssh** ключами.

**Как за пушить коммит?**
Делаем изменения далее:
git add *
git commit -m 'Your comment'
git push origin Your branch
